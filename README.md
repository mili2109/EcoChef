# 🌱 EcoChef

O EcoChef é uma plataforma web desenvolvida com Laravel que tem como objetivo incentivar a alimentação sustentável e o aproveitamento consciente dos alimentos. O sistema permite que usuários compartilhem receitas, descubram novos pratos e encontrem preparações de acordo com os ingredientes disponíveis, contribuindo para a redução do desperdício alimentar.

## 🚀 Funcionalidades

- Cadastro e autenticação de usuários
- Criação, edição e exclusão de receitas
- Visualização detalhada das receitas
- Classificação por tipo:
  - 🌱 Vegana
  - 🥬 Vegetariana
  - 🍖 Carnívora
- Filtro por ingredientes
- Filtro por categoria
- Dashboard com estatísticas da plataforma
- Contagem de receitas por tipo e categoria
- Contagem de usuários cadastrados
- Interface moderna e responsiva

## 🎯 Objetivo

O projeto foi desenvolvido como uma solução tecnológica alinhada à ODS 2 (Fome Zero e Agricultura Sustentável), promovendo o compartilhamento de receitas sustentáveis e incentivando práticas que reduzam o desperdício de alimentos.

## 🛠 Tecnologias Utilizadas

- Laravel
- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- Chart.js

## 📊 Dashboard

O sistema possui um painel administrativo que apresenta:

- Total de receitas cadastradas
- Total de usuários cadastrados
- Quantidade de receitas por tipo
- Quantidade de receitas por categoria
- Gráficos estatísticos interativos
